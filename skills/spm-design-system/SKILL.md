# SPM Design System — 视觉设计规范

## 设计工作流

```
1. 识别项目类型 → 匹配下方 18 种分类
2. 选择调性 (Pick ONE tone, commit fully)
3. 提取该类型的配色/字体/布局/反模式
4. 注入到每个 WBS 任务 Context Brief 的"设计约束"中
```

## 调性选择 (Do This First)

借鉴 `ultimate-frontend-design`：**选一个极端调性，温和设计一定平庸。**

| # | 调性 | 特征 | 适合 |
|---|------|------|------|
| 1 | **极致简约** | 黑白、大字、大量留白、原始边角 | 高端品牌、设计工作室 |
| 2 | **暗色专业** | 深底色、霓虹点缀、等宽字体 | SaaS、开发者工具、数据平台 |
| 3 | **柔和有机** | 圆角、暖色、自然曲线、手绘元素 | 健康、教育、生活方式 |
| 4 | **杂志编辑** | 强网格、衬线标题、留白即内容 | 媒体、内容平台、咨询 |
| 5 | **工业实用** | 功能优先、数据密集、无装饰 | 企业内部系统、运维面板 |
| 6 | **奢侈精炼** | 微妙动效、高级字体、克制配色 | 高端服务、珠宝、酒店 |
| 7 | **年轻潮流** | 大胆色块、动态排版、非对称 | 消费品牌、社交App、游戏 |
| 8 | **学术权威** | 纯白、衬线、≤3色、无动效 | 科研、政府、法律 |

每个项目类型会推荐**最适合的 1-2 个调性** + 不推荐的调性。

## 18 种项目类型

### 1. SaaS / 工具

| Token | 推荐 |
|-------|------|
| 🎭 调性 | **暗色专业** 或 **极致简约**；❌ 柔和有机、年轻潮流 |
| 主色 | 蓝 (`#2563EB`) 或深中性 (`#111827`) |
| 背景 | `#FAFAFA` 浅灰白，`#1A1A2E` 暗色 |
| 字体 | **BANNED**: Inter/Roboto/Arial → 用 Satoshi / Plus Jakarta Sans / DM Sans |
| 间距 | 紧凑 16px/24px/32px |
| 圆角 | 6-8px |
| 阴影 | `0 2px 8px rgba(0,0,0,0.08)` |
| 布局 | 侧边栏 + 主内容 |
| 动效 | 150ms ease，hover 上浮 2px |
| ❌ | 大色块、花哨渐变、emoji 图标、12px+ 圆角 |

## 通用反模式

| # | 反模式 |
